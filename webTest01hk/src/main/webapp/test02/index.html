제품관리

목록조회:
예1)http://localhost:8080/product/list
예2)http://localhost:8080/product/list?pageNo=페이지번호
예3)http://localhost:8080/product/list?pageNo=페이지번호&pageSize=페이지당 출력개수

제품추가: Create
http://localhost:8080/product/add?name=제품이름&qty=수량&mkno=제조사번호

제품조회: Retrieve or Read
http://localhost:8080/product/view?no=제품번호

제품변경: Update
http://localhost:8080/product/update?no=제품번호&name=제품이름&qty=수량&mkno=제조사번호

제품삭제: Delete
http://localhost:8080/product/delete?no=제품번호
